# Numpy Projects

## About
Collection of Numpy projects built during learning journey.

## Projects

### 1. Cricket Score Analyzer
Analyze cricket player scores using 2D arrays,
slicing, filtering and sorting.

### 2. Employee Weekly Performance Analyzer
Analyze employee weekly performance using numpy,
finding top/bottom performers and weekly averages.

### 3. Game Score Analyzer
Analyze random game scores using numpy arrays,
sorting and filtering.

### 4. Library Book Data Analyzer
Analyze library book data using numpy arrays,
filtering and aggregation.

## Tools Used
- Python
- Numpy
